# Special Syusseki System

## Requirement

- Ruby 2.4.x
- bundler

## Specifications

- Rails 5.1.2


### Specific Library

- haml-rails
- webcamjs

## Usage

### Prepare application

```
$ bundle install --without development test --path vendor/bundle
$ bundle exec rails db:migrate
```

### Run application

```
$ bundle exec rails s
```
